# style-guide.md

## 1) 视觉关键词
优雅、荒诞、克制、古典。

## 2) 配色建议
- 背景：#0F1014（深墨）
- 文本主色：#E8E3D6（羊皮纸白）
- 卡片底色：#E9DFC8（羊皮纸）
- 警示色：#B24A3A（暗红）
- 强调色：#8C7A4D（旧金）

## 3) 字体
- 英文：Playfair Display
- 中文：Noto Serif SC（思源宋体）
- 回退：serif

## 4) 排版
- 主内容宽度 720–820px
- 行高 1.6–1.8
- 标题使用小型大写或增加字距
- 每个回合信息块统一间距（8/12/16 体系）

## 5) 数值条样式
不要使用默认 progress。
建议“古老刻度尺”视觉：
- 细刻度背景（重复线性渐变）
- 前景条显示当前值
- 每条附数字与标签（Heaven/Hell/Stability/Pressure）

## 6) 动效
- 仅使用轻量过渡（150–250ms）
- 避免夸张弹跳
- 选项 hover 轻微提亮即可

## 7) 组件清单
- Header（标题+语言切换）
- Status Panel（四条数值）
- Event Card（标题/正文/选项）
- Archive Modal（记录阶段）
- Ending Screen（结局文本+重开按钮）

## 8) 文案风格
- 黑色幽默 + 宗教哲学 + 轻荒诞
- 不写说教句
- 通过后果表达观点
